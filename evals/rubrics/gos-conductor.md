# /gos Conductor Rubric

## Scoring (0-10)

### Context Loading (0-2)
- 0: No context gathered
- 1: Read scratchpad or git log, not both
- 2: Parallel gather: git log, scratchpad, memory, diff, active sessions

### Intent Clarification (0-2)
- 0: Jumped to execution without understanding goal
- 1: Restated goal but didn't bound scope or identify personas
- 2: Full elicitation scaled to complexity — restate, expand, bound, personas, confirm

### Decomposition (0-2)
- 0: No plan presented
- 1: Plan presented but missing dependencies or parallelism
- 2: Task graph with phases, dependencies, time estimates, presented for approval

### Execution (0-2)
- 0: Didn't execute or executed wrong tasks
- 1: Executed but missed parallel opportunities or had regressions
- 2: Clean execution with progress tracking, parallel where possible

### Reporting (0-2)
- 0: No summary
- 1: Summary but missing key findings or next steps
- 2: Consolidated report with executive summary, issues ranked, actions taken, remaining items
