# /evolve Rubric

## Scoring (0-10)

### Signal Analysis (0-2)
- 0: No signal data used
- 1: Read signals but didn't analyze patterns
- 2: Pattern analysis across signals — identified trends, root causes, command-level health

### Proposal Quality (0-2)
- 0: No upgrade proposals
- 1: Vague proposals ("improve X")
- 2: Specific proposals with: what changes, why, expected impact, implementation steps

### Implementation (0-2)
- 0: Proposals written but not implemented
- 1: Implemented but not tested
- 2: Implemented, tested (eval rubric if applicable), committed

### Learning Loop (0-2)
- 0: No memory or feedback updates
- 1: Updated one memory file
- 2: Updated relevant memories, promoted instincts, archived stale entries

### Metacognition (0-2)
- 0: No self-assessment
- 1: Listed what happened
- 2: Honest assessment of what gOS does well vs poorly, with confidence scores
