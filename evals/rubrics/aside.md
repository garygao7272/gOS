# /aside Rubric

## Scoring (0-10)

### Speed (0-3)
- 0: Took longer than the main task would have
- 1: Slow — researched extensively for a quick question
- 2: Reasonable — answered within a few tool calls
- 3: Fast — answered immediately or with one lookup

### Accuracy (0-3)
- 0: Wrong answer
- 1: Partially correct
- 2: Correct but missing nuance
- 3: Correct, concise, with relevant context

### Context Preservation (0-2)
- 0: Broke the main task flow — lost scratchpad state or switched modes
- 1: Returned to main task but lost some context
- 2: Clean aside — main task state fully preserved

### Brevity (0-2)
- 0: Rambling, off-topic tangents
- 1: Reasonable length but could be shorter
- 2: Concise — answered the question and nothing more
