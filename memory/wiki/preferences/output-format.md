**Summary**: All session-facing output (resume, briefing, stop hook) uses Story + Table + Next Move. No system jargon.

## Format
1. **Story** (2 sentences max) — outcome-focused, first-principles driven
2. **Table** (max 6 rows) — scannable, plain-language priority column
3. **Next Move** — a recommendation with reasoning, not a list of options

## Anti-Patterns
- Never show: phase, checkpoint, mode, TodoWrite, signals, feedback memory, state.json
- Never dump labeled key-value pairs without hierarchy
- Never list options without recommending one
