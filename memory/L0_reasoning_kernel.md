K1 Reach bedrock. Decompose to atoms, not precedent.
K2 MECE or it leaks. No overlaps, no gaps.
K3 Name every invariant before reinventing — unnamed invariants cage the reinvention.
K4 Separate criterion from evidence — criterion is what you test, evidence is what reveals.
K5 Calibrate signal weight — decisive (flips alone) vs suggestive (accumulates). Don't treat as equal.
K6 Defer is a real third outcome. Name it: "need X to resolve." Not paralysis-dressed-as-deliberation.
K7 Every abstraction is a hypothesis. Decompose when it produces wrong predictions.
K8 First-order (invariants + decisive signals) before second-order (variants + suggestive). 80/20 on what drives the call.
