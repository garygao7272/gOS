# bootstrap/templates/

Project-template files copied into new gOS projects by `install.sh --bootstrap`.

## Note on hooks

Hook scripts are **not** templated here. The canonical hooks live in
`.claude/hooks/` (project-level) and are distributed via the gOS plugin
(`gos-plugin-build/hooks/scripts/`). The legacy `bootstrap/templates/hooks/`
dir (21 stale files) was archived on 2026-04-17 to
`Archive/2026-04-17-tier1-cleanup/bootstrap-templates-hooks-content/` after
confirming no references in install.sh or elsewhere.
